# Руководство по конфигуратору дверей

## Обзор

Был создан полнофункциональный конфигуратор дверей с каталогом продукции согласно техническим требованиям. Система поддерживает все типы моделей дверей (Standard, Designer, Preset, Hidden) с правилами доступности и ограничениями.

## Структура данных

### Файлы данных
- `/data/door-configurator-data.ts` - центральный файл с данными моделей, цветов и опций

### Типы моделей
1. **Standard** - стандартные модели с полной кастомизацией
2. **Designer** - дизайнерские серии с ограниченным выбором
3. **Preset** - готовые пресеты без возможности изменения
4. **Hidden** - двери скрытого монтажа

### Категории цветов
- **Wood** - Текстуры дерева (Северный дуб, Темный дуб, Классик дуб и т.д.)
- **Solid** - Однотонные эмали (Молочный дуб, Дымчатый дуб, Хлопок белый и т.д.)
- **Concrete** - Бетон (Теплый бетон, графит, снежный и т.д.)
- **Decorative** - Декоративные покрытия (Шагрень, Ривьера и т.д.)
- **Glass** - Lacobel поверхности

### Категории опций
- **Hardware** - Фурнитура (замки, петли)
- **Glass** - Стекло (без рисунка, ромб, берн, версаль, шпион, зеркало)
- **Edge** - Кромка (АБС пластик, алюминий, бескромочная технология)

## Серии дверей

### L Series (27 моделей)
Стандартные модели серии L с артикулами VS-01–VS-15 (включая варианты) и полной кастомизацией

### LX Series (14 моделей)
Премиальная линейка серии LX с артикулами EG-01–EG-14

### S Series (9 моделей)
Современные модели серии S с артикулами SK-01–SK-15

### SH Series (8 моделей)
Специальная серия с артикулами GS-01–GS-07 и SD

### Classic Series (9 моделей)
Классические двери с артикулами CL-01/CL-02/CL-03 и их вариациями

### LINE Series (Designer)
Дизайнерские модели AL-01–AL-04 + комбинированные варианты SF-02V1–SF-04V4

### K NEO Series (Designer)
7 моделей дизайнерской серии с артикулами FR-D/FR-G/FR-O

### VERTA Series (Designer)
5 моделей серии с артикулами FS-02–FS-07

### SKY Series (Preset)
8 готовых пресетов без возможности изменения

### ID Series (Hidden)
Щитовые двери скрытого монтажа с артикулом PN

## Компоненты

### Основные компоненты
- `NewConfigurator.tsx` - главный компонент конфигуратора
- `CatalogNew.tsx` - каталог продукции

### Вспомогательные компоненты
- `ConfiguratorStepper.tsx` - индикатор шагов
- `DoorTypeBadge.tsx` - бейджи типов дверей
- `ModelSelection.tsx` - выбор модели с поиском и фильтрами
- `ColorSelection.tsx` - выбор цвета с группировкой по категориям
- `OptionsSelection.tsx` - выбор опций с группировкой
- `ConfiguratorPreview.tsx` - визуальный предпросмотр конфигурации

## Функциональность

### Пошаговый интерфейс
1. **Шаг 1: Модель** - выбор модели двери с поиском и фильтрацией по сериям
2. **Шаг 2: Цвет** - выбор цвета из доступных категорий
3. **Шаг 3: Опции** - выбор дополнительных опций (фурнитура, стекло, кромка)

### Правила доступности
- Disabled состояния для недосту��ных опций с тултипами-объяснениями
- Автоматический выбор цвета для preset моделей
- Блокировка цветов для комбинированных designer моделей
- Фильтрация опций по типу модели

### Бейджи
- **Дизайнерская серия** - для designer моделей
- **Готовый пресет** - для preset моделей
- **Скрытый монтаж** - для hidden моделей

### Интеграция с корзиной
- Полная совместимость с существующей системой корзины
- Автоматический расчет итоговой стоимости
- Сохранение всех параметров конфигурации

## Мобильная адаптация

Все компоненты полностью адаптированы для мобильных устройств:
- Адаптивная сетка
- Мобильная панель навигации внизу экрана
- Оптимизированные фильтры с overlay
- Touch-friendly элементы управления

## Цветовая схема

Интерфейс использует фирменную цветовую палитру:
- **Акцент**: #F7AB23 (золотистый)
- **Основной**: #15181C (темный)
- **Фон**: #FFFFFF (белый)
- **Вторичный текст**: #7B7E8B (серый)

## Использование

### Запуск каталога
```typescript
onNavigate('catalog')
```

### Запуск конфигуратора
```typescript
onNavigate('configurator')
```

### Добавление в корзину
Конфигуратор автоматически добавляет настроенную дверь в корзину и переводит на страницу корзины.

## Расширение

### Добавление новой модели
Добавьте новый объект в массив `doorModels` в `/data/door-configurator-data.ts`:

```typescript
{
  id: 'new-model',
  name: 'Новая модель',
  series: 'Новая серия',
  type: 'standard',
  basePrice: 20000,
  availableColors: ['wood-1', 'solid-1', ...],
  availableOptions: ['hw-1', 'edge-1', ...]
}
```

### Добавление нового цвета
Добавьте новый объект в массив `colorOptions`:

```typescript
{
  id: 'new-color',
  name: 'Новый цвет',
  category: 'wood',
  price: 1500
}
```

### Добавление новой опции
Добавьте новый объект в массив `configOptions`:

```typescript
{
  id: 'new-option',
  name: 'Новая опция',
  category: 'hardware',
  price: 2000,
  description: 'Описание опции'
}
```
